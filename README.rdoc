== README

This is a sample webscraper app pulling up-to-date data from Craigslist (www.craigslist.com) on rental accomodation in Brooklyn, New York, USA. Data is being scraped by using the 3Taps API from www.3taps.com. 

If you wish to view this app on Heroku, please check out: https://craigslist-rental-scraper.herokuapp.com
